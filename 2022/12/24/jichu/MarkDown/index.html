# MarkDown 学习

# 标题

# 三级标题

# 四级标题

层级标题的快捷方式由 #+ 空格 + 标题名称 表示，回车键显示。

一级标题用一个 #

二级标题用两个 ##

以此类推，Typora 最高只能由六个层级标题

# 字体

严于律己，宽以待人

严于律己，宽以待人

严于律己，宽以待人

严于律己，宽以待人

在两端加上 * 号，用一个 ***** 表示斜体；两个 ****** 表示黑体；三个 ******* 表示黑体和斜体的组合。

在两端加上两个 ~~ 表示删除线

# 引用

好好学习，天天向上

用英文状态的 >+ 空格 表示

# 分割线

分割线有两种表示方式

用三个 — 回车表示；用三个 ******* 回车表示

# 图片

![图片 1](C:\Users\HP\Pictures\Saved Pictures\1.jpg)

图片用 **![图片名] (图片路径) ** 来表示

# 超链接

点击跳转到百度

超链接用 [文字] (超链接地址) 表示

# 列表

  1. 好好
  2. 学习
  • 天天
  • 向上

有序列表用 1. + 空格 表示

无序列表用 - + 空格 表示

# 表格

名字 | 性别 | 年龄

–|--|–|

张三 | 男 | 12

在源代码模式中将空格删除，退出源代码模式即可

名字 性别 年龄
张三 男 12

# 代码

代码写法由 ``` 回车表示，也可以加上代码类型再回车。
